MailChimp Credentials
Please make sure the boxes on this page show the parameters necessary to send MAPS Newsletters through MailChimp:

List ID:  

You find this ID after logging in to MailChimp. Select "Lists" in the grey bar on top, and navigate to the list you want to use. Select "List Name and Defaults" from the "Settings" menu, and copy the red alphanumeric string on this page.


API Key:  

You can see your API key (or generate a new one) by logging on to MailChimp. Use the drop-down menu in the top right corner that is labelled with your Mailchimp account name, and select Accounts. Navigate to Extras, API keys.
